### Option 2: From source

Requires [Node.js](https://nodejs.org) and [Git](https://git-scm.com).

```bash
git clone https://github.com/soundgrail/music-theory-app.git
cd music-theory-app
npm install
npm start
```

## ✨ Features

- Learn chords, scales, and progressions interactively
- Practice ear training and music theory
- Free and works right in your browser
